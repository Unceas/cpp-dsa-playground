# C++ DSA Playground

A structured repository to practice **Data Structures & Algorithms in C++**, with clean implementations and detailed complexity analysis.

## Folder Structure

- `Arrays/`
  - `max_element.cpp` – Find maximum element in an array
  - `linear_search.cpp` – Search for an element linearly
  -   - `two_sum.cpp` - Find indices of two numbers whose sum equals target
- Linear Search
  - Binary Search (Iterative & Recursive)
  - Binary Search using STL
  - - "Strings/"
  - `reverse_string.cpp` - Reverse a string using two-pointer technique
  -   - "palindrome_check.cpp" - Palindrome Check
      -   - "valid_palindrome.cpp" - Valid Palindrome (Ignoring non-alphanumeric characters)
          -   - "first_non_repeating_char.cpp" - First Non-Repeating Character
## Features

- Clean, beginner-friendly C++ code
- Time and space complexity for each solution
- Uses standard C++ and STL where helpful

## How to Run

```bash
g++ Arrays/max_element.cpp -o max_element
./max_element

g++ Arrays/linear_search.cpp -o linear_search
./linear_search
```
