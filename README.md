# C++ DSA Playground

A structured repository to practice **Data Structures & Algorithms in C++**, with clean implementations and detailed complexity analysis.


## Features

- Clean, beginner-friendly C++ code
- Time and space complexity for each solution
- Uses standard C++ and STL where helpful

## Planned Topics
- Arrays
- Strings
- Recursion
- Linked List
- Stack & Queue
- Trees
- Graphs
- Backtracking
- Dynamic Programming

## How to Run

```bash
g++ Arrays/max_element.cpp -o max_element
./max_element

g++ Arrays/linear_search.cpp -o linear_search
./linear_search
```
